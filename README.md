# The Website That Hates You

An intentionally awful interactive website created for the Ugly Web Challenge 2026.

## Live Website
https://thewebsitethathatesyou.netlify.app

## Concept
This website is designed to be intentionally hostile and frustrating. Every click makes the experience worse through misleading navigation, fake rewards, painful visuals, pointless interactions, and escalating chaos.
